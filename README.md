# Chinese Learning Plan Website

This repository now hosts a static GitHub Pages site for a complete Chinese learning plan designed for international beginners.

## Content

- 24-week Chinese learning roadmap
- Four study phases
- 12 chapters with concrete learning goals, lesson topics, and output tasks
- A standard chapter structure for vocabulary, sentence patterns, grammar, speaking, reading, writing, and review
- Milestones for each learning stage
- A teaching note explaining that later instruction can be translated and explained primarily in English

## Files

- `index.html` — main learning plan page
- `styles.css` — page styling

## Usage

Open the site locally in a browser or publish it with GitHub Pages to share the learning plan online.

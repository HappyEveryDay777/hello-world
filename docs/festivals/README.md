## 关于本课程

本系列以**中华文化传承者**的视角，带你走进中国传统节日的世界。

中国的传统节日，不只是吃饭、放假、贴春联那么简单。每一个节日背后，都有几千年的历史积淀——农耕文明的时间智慧、祭祖礼仪的家族情感、季节自然的生命感悟，以及代代相传的民间故事。

本课程希望做到：用清晰、生动的语言，帮助每一位读者——无论大人还是孩子——真正理解每个节日"为什么这样过"，而不只是知道"这个节日吃什么"。

## 课程目标

学完本系列，你将能够：

1. 说出每个传统节日的时间（农历日期或节气）、来源和核心意义
2. 理解节日与历法（节气、月相）之间的深层关联
3. 了解节日中的主要习俗及其文化含义
4. 区分历史记载与民间传说，理解节日习俗如何随时间演变
5. 找到在现代生活中传承节日文化的实际方式

## 一句话理解每个节日

如果一定要用一句话概括每个节日的文化精髓：

- **春节**：全家人用整整一年的分别，换来的那一顿年夜饭。
- **元宵节**：年还没过完，先把灯点亮，把夜晚变成白昼。
- **清明节**：我们不只活在当下，我们也住在那些离开的人的心里。
- **端午节**：一根粽绳，系住的不只是糯米，还有对忠诚与热爱的永久纪念。
- **七夕节**：爱情最美的样子，不是天天在一起，而是银河两岸的守望。
- **中元节**：对逝者的思念，不需要藏起来，它可以是一盏灯，一炷香。
- **中秋节**：无论身在何处，月亮都一样圆，那是我们共同的家。
- **重阳节**：登上高处，看见了更远的地方，也看见了父母白了的鬓角。
- **腊八节**：冬天最寒冷的时候，一碗热粥暖胃，也暖年。
- **小年**：年，终于来了，先送走灶王爷，把家打扫干净，等着它。

## 关于节日的时间说明

本系列涉及的节日，时间来源分两类：

1. **农历日期**（如正月初一、七月初七）：这类节日在公历上的日期每年不同
2. **节气时间**（如清明节气）：这类节日在公历上的日期每年基本固定（通常相差1—2天）

每篇文章都会注明具体时间类型，并给出大概的公历对应月份，方便读者提前计划。

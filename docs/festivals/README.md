## 这门课是关于什么的

中国传统节日，是历法与文化最美丽的交汇点。

每一个节日，都不是随机选出的日子——它们扎根于月亮的圆缺、太阳的轨迹、季节的更替。春节在冬末的新月，元宵在正月的满月，中秋在秋天最圆的月亮之下……节日的时间选择，本身就是一种诗意。

这门课程将带你跟随一年的脚步，走过春夏秋冬，走过团圆与思念，走过祭祖与庆丰，走过驱邪与迎新——走过中国人对时间最深情的诠释。

## 为什么学习传统节日很重要

传统节日，是中华文化的"活体百科全书"：

- 🧧 **传承历史**：每个节日背后都有历史故事（真实的与传说的），是了解中国历史的窗口
- 🌙 **理解历法**：节日的时间逻辑，是理解传统历法最直观的方式
- 👨‍👩‍👧 **凝聚家庭**：节日是家庭最重要的集结时刻，是亲子文化教育的最佳场合
- 🌾 **连接自然**：节日的食物、习俗往往与季节直接对应，体现"天人合一"的生活哲学
- 🌍 **走向世界**：中国传统节日正在成为全球华人社区乃至世界的文化节庆

## 节日与历法的关系

学完本系列，你会发现一个规律：绝大多数中国传统节日，都选在了历法中具有特殊意义的时间节点：

| 历法节点 | 对应节日 |
|---------|---------|
| 正月初一（新月） | 春节 |
| 正月十五（满月） | 元宵节 |
| 清明节气 | 清明节/寒食节 |
| 五月初五（仲夏） | 端午节 |
| 七月初七 | 七夕节 |
| 七月十五（满月） | 中元节 |
| 八月十五（满月） | 中秋节 |
| 九月初九（重阳数） | 重阳节 |
| 腊月初八 | 腊八节 |
| 腊月廿三/廿四 | 小年 |

如需深入理解节日与历法的关系，请参阅配套的[中国历法文化课程](../chinese-calendar/README.md)。

## 关于地域差异

中国幅员辽阔，各地节日习俗存在丰富的地域差异。本系列以中华文化主流习俗为主线，同时会在各课中提示主要地区差异。

几个典型例子：
- **春节**：北方吃饺子，南方吃年糕/汤圆
- **小年**：北方腊月廿三，南方腊月廿四
- **端午粽子**：北方偏甜（枣粽），南方偏咸（肉粽）
- **中元节**：南方仪式感更强，有放河灯等习俗

这些差异，不是"哪个才是正确的"，而是同一文化在不同土地上生长出的不同面貌，都值得尊重和了解。

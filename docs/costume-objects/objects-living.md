# 器物第六章 · 日常生活器具
## 柴米油盐里的美学

**时代跨度**：各历史时期均有涉及，重点明清至今  
**核心主题**：漆器、铜镜、灯具、竹器、茶器、香器等日用器物  
**关键词**：漆器、铜镜、油灯、竹编、茶器（茶盏茶壶）、香器

## 【开篇故事】

宋代的一个普通秋日下午，一位老妇人坐在院子里，手里捧着一只深棕色的漆木茶碗，慢慢地啜饮着热茶。

茶碗并不华贵，但漆面光滑如镜，映出她老花眼中摇曳的光线。那只碗，是她的母亲传给她的，母亲的母亲传给她的，已经不知传了几代。岁月在漆面上留下了细微的纹路，每一道痕迹，都是时光的印章。

喝完茶，她小心地用布擦干茶碗，放回木匣，轻轻合上。

在中国传统文化里，器物不只是用来盛放、承载的工具——它们是生活的见证者，是家族记忆的容器，是日常之美最朴素的载体。

这就是中国人对待日常器物的方式：**用心、爱惜、传承**。

## 【漆器——用时间熬出来的美丽】

### 漆器的历史

中国是世界上最早使用天然漆（大漆，来自漆树）的国家，漆器历史超过8000年（浙江余姚河姆渡遗址出土了约前6000年的漆碗）。

天然漆是漆树分泌的汁液，采集后经过加工，可以涂在木、竹、皮革等材质上，形成光滑、坚硬、防水、防腐的保护层。

### 漆器工艺的类型

| 工艺名称 | 特点 |
|---------|------|
| **彩绘漆器** | 在漆面上用彩色漆绘制图案，色彩鲜艳 |
| **雕漆（剔红等）** | 在厚厚的漆层上雕刻纹样，立体感强 |
| **描金漆器** | 用金粉在漆面上描绘图案 |
| **螺钿漆器** | 将贝壳薄片镶嵌在漆面上，闪烁珠光 |
| **犀皮漆** | 利用漆液流动形成自然纹样，如木纹或大理石纹 |

### 马王堆漆器——汉代漆器的传奇

长沙马王堆汉墓出土了500多件漆器，是迄今出土规模最大、保存最完好的汉代漆器群。其中包括各种食盘、耳杯（有双耳的浅杯，用于饮酒）、箱奁（梳妆盒）、棺椁等，大多以红黑两色为主，绘有云气纹和神兽纹，历经2000年，颜色依然鲜艳如新。

## 【铜镜——映照时光的神奇圆镜】

在玻璃镜普及之前，中国人用铜镜照面已有4000多年历史。

### 铜镜的基本形制

铜镜通常是圆形的铜片，一面磨光（照面），一面有精美纹样和钮（中央凸起的小圆，用于穿绳提镜）。

### 各时期铜镜纹样演变

| 时代 | 代表纹样 | 风格特点 |
|------|---------|---------|
| 战国 | 蟠螭纹、山字纹 | 精细繁密 |
| 汉代 | 四神规矩纹（TLV纹）、铭文纹 | 宇宙观鲜明 |
| 唐代 | 花鸟纹、瑞兽葡萄纹、菱花形 | 华丽繁盛 |
| 宋代 | 简单花卉纹 | 素雅精洁 |

**唐代"瑞兽葡萄镜"**是铜镜史上最精美的品种之一：镜背以高浮雕手法铸造出葡萄藤蔓和各种瑞兽，繁密有致，葡萄这个来自西域的植物，也借助铜镜出现在千家万户的梳妆台上——文化融合如此悄无声息地发生在日常器物上。

## 【灯具——夜晚里的温暖时光】

### 中国古代灯具的演变

古代灯具经历了从**陶灯、铜灯**到**瓷灯、纸灯笼**的演变：

**战国秦汉：铜灯的黄金时代**  
汉代出土了大量精美的铜灯，其中最著名的是：

**长信宫灯**（西汉，现藏河北博物院）：
- 是一位宫女跪坐、手持灯盏的铜制器物
- 宫女的袖子成为灯的挡风板，可以调节光线方向
- 中空的身体连通袖管，将油烟导入宫女体内再从袖口排出——这是世界上最早有记载的**环保灯具设计**
- 宫女的神情安详而略带忧郁，是汉代青铜艺术中最有人情味的作品

**灯笼的文化意义**：  
中国灯笼有超过2000年历史，是节庆文化最重要的器物之一：
- 元宵节（正月十五）是"灯节"，家家悬灯
- 红灯笼象征喜庆、平安、驱邪
- 走马灯是中国古代最精妙的机械玩具之一——热气推动灯内的纸片旋转，形成剪影故事

---

## 【竹器——来自植物的智慧】

中国是竹子的故乡，也是竹器文化最发达的国家。竹子轻盈、坚韧、环保，被加工为：

- **竹编器物**：竹篮、竹箱、竹帘、竹席，涵盖生活的方方面面
- **竹制文具**：竹简（古代书写材料）、笔杆、笔架
- **竹制家具**：竹椅、竹床，在南方农村至今仍常见
- **竹制乐器**：笛子、箫、笙——中国最重要的管乐器，都是竹制的

竹编工艺在中国非遗名录中占有重要位置，浙江东阳、四川自贡的竹编，能织出精密如布的图案，令人叹为观止。

## 【茶器——喝茶不只是喝茶】

茶器是中国日常生活中文化内涵最丰富的器物类别之一。

### 茶器的主要种类

| 茶器 | 功能 | 著名品种 |
|------|------|---------|
| **茶壶** | 泡茶 | 紫砂壶（宜兴）、瓷壶 |
| **茶盏/茶碗** | 饮茶 | 建盏（福建建窑）、汝瓷茶盏 |
| **茶托（盏托）** | 托着茶盏，防烫手 | 宋代流行 |
| **茶罐** | 储存茶叶 | 锡罐、瓷罐 |
| **茶则（茶量勺）** | 量取茶叶 | 竹制、木制 |

**宜兴紫砂壶——茶器之王**：  
江苏宜兴产的紫砂壶，是中国最著名的茶具。紫砂是一种特殊的泥料，烧制后呈棕红色，质地透气而不渗水，泡茶能保留茶香而不夺味。

一把好的紫砂壶，经过数年使用，壶体会慢慢被茶水浸润，颜色越来越深，光泽越来越内敛——这个过程叫"养壶"，喝的是茶，养的是时间，也是一种生活美学。

**建盏——黑瓷中的宋代美学**：  
宋代文人喝茶（斗茶）流行使用福建建窑烧制的**建盏**——黑色的茶碗，釉面有"兔毫"（细如兔毛的金色线纹）或"油滴"（如滴落的油珠状斑点）的自然纹样。黑色的背景上，白色的茶沫清晰可见，这是宋代"斗茶"（比较茶的品质）的最佳视觉背景。

---

## 【香器——焚香的优雅仪式】

**焚香**是中国传统生活中极为重要的文化仪式，与茶、花、琴并称"文人四艺"。

### 主要香器种类

| 香器 | 功能 |
|------|------|
| **香炉** | 焚烧块状香或线香，多为铜制或瓷制 |
| **香插** | 插放线香 |
| **香盒（香盒）** | 储存香料 |
| **香筒** | 竹制或木制，线香从顶部开孔慢慢散出 |

**宣德炉**（明代宣德年间铸造的铜香炉），是中国香器史上最著名的品种。宣德炉以特殊配方的铜合金铸造，颜色温润，经过岁月侵染后呈现出棠梨色或佛金色，据说仿制了无数，但真品极难鉴别，是收藏界最玄妙的领域之一。

## 【课程小结】

✅ 日常器物是文化最日常也最深刻的载体，涵盖生活方方面面

## 🎓 器物篇总结：物以载道

我们走过了青铜、瓷器、玉器、家具、文房、日常生活器具这六大领域，让我们回顾一下中国器物文化的核心精神：

| 器物类型 | 文化核心 |
|---------|---------|
| 青铜器 | 礼制秩序与权力象征 |
| 瓷器 | 泥与火的极致之美 |
| 玉器 | 君子品德的物质化身 |
| 家具 | 榫卯智慧与生活哲学 |
| 文房四宝 | 文人精神世界的物质基础 |
| 日常生活器具 | 生活美学与匠心传承 |

每一件器物背后，都有一段历史、一个故事、一种中国人对生活的理解与热爱。

"**物以载道**"——器物承载着道理，承载着文化，承载着一代代中国人对美好生活的深情向往。

这就是中国器物文化最深沉、最温润的力量。

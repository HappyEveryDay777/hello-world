# 第07课：天干地支入门

**所属系列**：中国历法课程  
**建议阅读时长**：约18分钟  
**适合人群**：亲子共读、文化爱好者

---

## 【开篇引入】

"今年是甲辰年。"

"这场大地震发生在戊戌年。"

"他是庚子年生人。"

"今天是壬申日。"

在中国的文献、历史记载、新闻报道和日常对话中，这类表达随处可见。它们用的都是同一套符号系统——**天干地支**。

这套系统，是中国历法中最独特、最有文化内涵的部分之一。它不只是一套计时工具，更是中国传统哲学、宇宙观和生命观的具体体现。

今天这一课，我们从零开始，把天干地支讲清楚。

---

## 【核心知识】

### 一、天干：十个天上的符号

**十天干**，按顺序是：

> **甲（jiǎ）、乙（yǐ）、丙（bǐng）、丁（dīng）、戊（wù）、己（jǐ）、庚（gēng）、辛（xīn）、壬（rén）、癸（guǐ）**

这十个字，每循环一轮，叫做"一干"，周期是10。

**天干的含义**

天干不只是编号，每个字都有原始含义，与万物生长周期有关：

| 天干 | 原义 | 象征含义 |
|-----|------|--------|
| 甲 | 种子破壳而出 | 生命的开始，萌芽 |
| 乙 | 幼苗弯曲生长 | 柔软的成长 |
| 丙 | 草木茂盛，阳光充足 | 旺盛，明亮 |
| 丁 | 壮实的成年植物 | 强健，充实 |
| 戊 | 茂盛顶点 | 鼎盛，高峰 |
| 己 | 开始转化、结果 | 收拢，自省 |
| 庚 | 果实成熟，收获 | 刚强，收获 |
| 辛 | 新的辛苦，脱变 | 辛辣，更新 |
| 壬 | 内蕴生机，静待 | 孕育，潜藏 |
| 癸 | 种子再次孕育，轮回 | 终结与新生 |

可以看出，十天干描绘了一个生命的完整周期：从萌芽（甲）到成长（乙丙丁）到鼎盛（戊）到收获（己庚辛）到归藏（壬癸），再回到新的甲——这是一个周而复始的生命轮回。

**天干与阴阳五行**

十天干还与中国传统哲学的核心概念"阴阳五行"相结合：

| 五行 | 阳干 | 阴干 |
|-----|-----|-----|
| 木 | 甲 | 乙 |
| 火 | 丙 | 丁 |
| 土 | 戊 | 己 |
| 金 | 庚 | 辛 |
| 水 | 壬 | 癸 |

奇数序号（甲丙戊庚壬）为阳干，偶数序号（乙丁己辛癸）为阴干。五行两两配对，天干就与中国传统的宇宙观完美整合。

### 二、地支：十二个地上的符号

**十二地支**，按顺序是：

> **子（zǐ）、丑（chǒu）、寅（yín）、卯（mǎo）、辰（chén）、巳（sì）、午（wǔ）、未（wèi）、申（shēn）、酉（yǒu）、戌（xū）、亥（hài）**

这十二个字，每循环一轮，叫做"一支"，周期是12。

**地支与时辰**

十二地支最早用于**记录时辰**（一天24小时分成12段，每段2小时）：

| 地支 | 时辰 | 对应现代时间 |
|-----|-----|-----------|
| 子时 | 夜半 | 23:00—01:00 |
| 丑时 | 鸡鸣 | 01:00—03:00 |
| 寅时 | 平旦 | 03:00—05:00 |
| 卯时 | 日出 | 05:00—07:00 |
| 辰时 | 食时 | 07:00—09:00 |
| 巳时 | 隅中 | 09:00—11:00 |
| 午时 | 日中 | 11:00—13:00 |
| 未时 | 日昳 | 13:00—15:00 |
| 申时 | 哺时 | 15:00—17:00 |
| 酉时 | 日入 | 17:00—19:00 |
| 戌时 | 黄昏 | 19:00—21:00 |
| 亥时 | 人定 | 21:00—23:00 |

这就是中文中"午时"（正午12点）、"子夜"（午夜12点）等词语的来源。

**地支与月份**

地支还用于标记月份（注意这里的地支月与生肖年不同）：

| 月份 | 对应地支 | 对应节气区间 |
|-----|---------|-----------|
| 正月（1月） | 寅月 | 立春—惊蛰 |
| 二月 | 卯月 | 惊蛰—清明 |
| 三月 | 辰月 | 清明—立夏 |
| 四月 | 巳月 | 立夏—芒种 |
| 五月 | 午月 | 芒种—小暑 |
| 六月 | 未月 | 小暑—立秋 |
| 七月 | 申月 | 立秋—白露 |
| 八月 | 酉月 | 白露—寒露 |
| 九月 | 戌月 | 寒露—立冬 |
| 十月 | 亥月 | 立冬—大雪 |
| 冬月（11月） | 子月 | 大雪—小寒 |
| 腊月（12月） | 丑月 | 小寒—立春 |

注意：地支月是按节气划分的，与农历月份有一定差异。

### 三、干支组合：六十年一个大循环

天干（10个）和地支（12个）依次配对，10和12的最小公倍数是60，因此干支组合总共有**60种**，称为"六十甲子"。

配对规则：阳干配阳支，阴干配阴支，因此不是所有天干地支都能组合（例如，甲是阳干，只能与子、寅、辰、午、申、戌这六个阳支配对）。

**六十甲子表（部分）：**

| 序号 | 干支 | 序号 | 干支 | 序号 | 干支 |
|-----|-----|------|-----|------|-----|
| 1 | 甲子 | 21 | 甲申 | 41 | 甲辰 |
| 2 | 乙丑 | 22 | 乙酉 | 42 | 乙巳 |
| 3 | 丙寅 | 23 | 丙戌 | 43 | 丙午 |
| 4 | 丁卯 | 24 | 丁亥 | 44 | 丁未 |
| 5 | 戊辰 | 25 | 戊子 | 45 | 戊申 |
| … | … | … | … | 60 | 癸亥 |

六十年完成一个大循环，称为一个"甲子"。"年过花甲"（60岁以上），就是指走完了一个完整的干支循环。

### 四、干支的用途

干支被用来标记：
- **年**（干支纪年）：如甲辰年（2024年）、壬寅年（2022年）
- **月**（干支纪月）：结合天干和地支记录月份
- **日**（干支纪日）：从殷商时代就开始使用，是目前已知世界上连续使用时间最长的历法记录体系，三千多年从未间断
- **时**（干支纪时）：记录每天的十二个时辰

一个完整的传统时间表达，包含年、月、日、时四个干支，合称"**四柱**"（命理学中用于推算命运的依据）。

## 【历史与演变】

**殷商时代**：甲骨文中大量出现干支纪日记录，是干支系统有文字记载的最早证据。殷商祭祀的日期、战争出发的日期，都用干支标记，形成了连续三千多年的"干支纪日"传统。

**汉代**：干支开始广泛用于纪年，并与五行、生肖等体系融合，形成了完整的干支文化体系。

**唐宋以来**：干支与命理学（四柱推命、八字算命）的结合日益深入，成为中国传统命理文化的核心框架。

今天，干支年份仍然出现在春节对联、年历印刷、传统婚书等场合，是活态的文化遗产。

---

## 【文化意义】

天干地支不只是历法符号，它是中国传统宇宙观的语言。

在中国人的思维里，时间不是线性的（从过去流向未来，一去不返），而是**周期性循环**的：十天、十二天一小循环，六十年一大循环，生命和历史在其中循环往复。这与西方历史观中的"直线进步"形成了鲜明的对比。

干支也是中国传统命理文化（俗称"算八字"）的基础。每个人出生的年、月、日、时各对应一个干支，合成"八字"，命理师据此推断一个人的性格、运势。这种信仰是否科学，可以见仁见智，但它深深植根于中国传统文化，影响着无数人的婚嫁、择业、出行决策。

## 【亲子共学问题】

1. **查一查**：今年是什么干支年？对应的天干是什么五行属性？
2. **记忆挑战**：试着背出十天干和十二地支各自的顺序。
3. **推算练习**：如果今年是甲辰年（2024年），那么60年前是哪一年？那一年发生了什么历史事件？

## 【本课小结】

- 十天干（甲至癸）和十二地支（子至亥）是中国传统历法的基础符号系统。
- 干支组合产生60个不重复的组合，称为"六十甲子"，用于纪年、纪月、纪日、纪时。
- 干支与阴阳五行结合，构成中国传统宇宙观的基础语言。
- 干支纪日是世界上连续使用时间最长的历法记录体系，三千多年未曾中断。

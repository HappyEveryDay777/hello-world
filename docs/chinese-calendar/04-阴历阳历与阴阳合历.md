# 第04课：阴历、阳历与阴阳合历

**所属系列**：中国历法课程  
**建议阅读时长**：约15分钟  
**适合人群**：亲子共读、文化爱好者

---

## 【开篇引入】

每逢春节，总有人发出一个灵魂疑问："今年春节怎么这么早/晚？"

2023年的春节在1月22日，2024年在2月10日，2025年在1月29日——公历日期每年都不同，有时候相差将近三周。

如果有人追问："那明年春节是几月几日？"，很多人不得不掏出手机查一下，因为不像圣诞节永远是12月25日，春节的公历日期真的"不固定"。

但换一种说法：**春节永远是农历正月初一**。这一点，从来没有变过，也永远不会变。

今天这一课，我们就来彻底说清楚：阴历、阳历、阴阳合历，三者的区别和逻辑，以及农历究竟是哪一种？

---

## 【核心知识】

### 一、阳历：跟着太阳走

**阳历的核心原理**

阳历（Solar Calendar），以地球绕太阳公转一圈的时间为"一年"。这个时间，天文学上叫做**回归年**，约等于**365.2422天**。

公历（格里高利历）的处理方法：
- **平年**：365天
- **闰年**：366天（4月增加2月29日）
- **规则**：能被4整除的年份是闰年，但能被100整除而不能被400整除的年份不是闰年。

例如：2000年是闰年（能被400整除），1900年不是闰年（能被100整除但不能被400整除），2024年是闰年（能被4整除且不是整百年）。

**阳历的特点**

优点：
- 与季节**完美对应**，每年春分永远在3月19-21日，夏至永远在6月20-22日
- 规则简单，计算方便
- 已成为国际通用标准

缺点：
- 月份与月相**完全无关**。公历的"2月"、"6月"等，与月亮的圆缺没有任何对应关系
- 每个月的天数（28、29、30、31天）不统一，容易让人困惑

### 二、阴历：跟着月亮走

**阴历的核心原理**

阴历（Lunar Calendar），以月亮朔望一次的时间为"一个月"。朔望月平均约**29.53天**。

12个朔望月 = 12 × 29.53天 ≈ **354.37天**

但一个回归年是365.2422天，两者相差约**11天**。

纯阴历的处理方法：不管这个差，就是12个月为一年，每隔几十年，年份和季节之间的关系才循环一次。

以伊斯兰历为例：它是目前世界上使用最广泛的纯阴历，每年只有354天或355天，每年比太阳年少约11天，所以伊斯兰历的重要节日（如斋月开始）在公历上每年提前约11天，约33年完成一次"整圈漂移"。

**阴历的特点**

优点：
- 月份与月相**完美对应**：每月初一是新月，每月十五（或十六）必然是满月
- 对没有任何仪器的普通人来说，很容易通过观察月亮来判断日期

缺点：
- 季节**不断漂移**：今年斋月在夏天，几年后可能在春天，再过若干年又在冬天
- 对农业生产**极不友好**：无法指导依赖季节变化的农耕活动

### 三、阴阳合历：兼而有之

**阴阳合历的核心原理**

阴阳合历（Lunisolar Calendar），试图同时追踪月亮和太阳，做到：
- 每个月的初一是新月，十五（或十六）是满月（像阴历一样）
- 一年的平均长度等于一个回归年（像阳历一样），季节不漂移

关键问题：12个朔望月（354天）比一个回归年（365天）少11天，怎么弥补？

答案：**置闰月**——每隔约2—3年，在某个月后面加一个"闰月"（比如闰四月、闰五月），让这一年有13个月，约385天，从而补偿积累的差距。

**19年7闰法**

中国历法采用的"置闰"规律，叫做**19年7闰法**（亦称"章法"）：在每19个农历年中，设置7个闰年（每个闰年有13个月），这样19个农历年的总天数与19个太阳年非常接近（误差不到2小时）。

这个规律大约在春秋战国时期已经被中国天文学家发现，与古希腊天文学家默冬（Meton）发现的"默冬周期"不谋而合。

**具体的置闰规则**

那么，闰月应该加在哪里？中国历法用了一个精妙的规则：
- 以"含有节气中气（偶数节气，如雨水、春分、谷雨等）"来定义"正常月份"
- 如果某个农历月内**没有中气**（这种情况在19年周期中会出现7次），则该月定为闰月

这个规则既保证了节气不漂移，又能决定闰月的位置，形成了一套自洽的闰月计算体系。

### 四、农历的真实面目

现在，我们可以给出一个完整的描述：

> **中国农历，是一套以朔望月为基础的月份系统，通过19年7闰的置闰规则校正年度误差，同时内置二十四节气系统精确标注太阳位置，配合干支纪年体系，形成的阴阳合历。**

关键要点：

| 维度 | 农历的规定 |
|------|----------|
| 一个月 | 以新月（朔日）到下一个新月，约29或30天 |
| 一年 | 通常12个月（平年），每隔2—3年有13个月（闰年） |
| 季节标记 | 二十四节气（完全按太阳位置计算，与月相无关） |
| 纪年方式 | 干支（甲子、乙丑……）+ 生肖纪年 |

## 【历史与演变】

中国农历的置闰规则，经过了漫长的演进。

**先秦时代**：已经有了粗略的置闰概念，但方法不统一，各诸侯国使用不同的历法。

**汉代《太初历》**（公元前104年）：确立了19年7闰的标准规则，以冬至为历法计算起点，以正月为岁首，奠定了中国历法的基本框架，这个框架沿用至今已超过两千年。

**南北朝—隋唐时代**：历法计算精度持续提升，逐渐引入更精确的天文常数，处理了太阳和月亮运动并不完全均匀的问题（即"不均匀运动"修正）。

**元代《授时历》**：由郭守敬主持编制，采用了当时世界最高精度的天文参数，将回归年长度定为365.2425天，与现代测量值几乎一致，成为中国历史上精度最高的历法。

## 【文化意义】

农历月份的名称在中国文化中有着独特的美感和丰富内涵：

| 农历月份 | 别称 | 含义 |
|---------|-----|------|
| 正月 | 元月、端月 | 岁之始，万象更新 |
| 二月 | 如月、杏月 | 杏花渐开，春意萌动 |
| 三月 | 桃月、暮春 | 桃花盛开，春色烂漫 |
| 四月 | 梅月、清和 | 梅子初成，天气清和 |
| 五月 | 蒲月、仲夏 | 菖蒲盛开，午时正气 |
| 六月 | 荷月、伏月 | 荷花盛放，三伏天 |
| 七月 | 巧月、兰月 | 七夕乞巧，兰花芬芳 |
| 八月 | 桂月、仲秋 | 桂花飘香，月圆中秋 |
| 九月 | 菊月、暮秋 | 菊花盛开，重阳时节 |
| 十月 | 良月、小阳春 | 天高气爽，短暂回暖 |
| 十一月 | 葭月、冬月 | 芦苇吐穗，冬日渐深 |
| 十二月 | 腊月 | 年末祭祀，腊八节 |

这些别称，把一年十二个月与自然景物、节庆习俗紧密联结，使历法本身成为一首关于时间与生命的诗。

## 【亲子共学问题】

1. **计算练习**：如果今年农历有一个闰月（比如闰二月），那今年农历有几天？（提示：13个月，每月约29.5天。）
2. **概念辨析**：有人说"今天是阴历初五"，这种说法准确吗？为什么？
3. **文化探索**：查一查你的农历生日，找到那个月份的别称，思考一下那个月份在大自然中有什么特别的景象。

## 【本课小结】

- 阳历以太阳年（365.2422天）为基础，季节稳定，但月份与月相无关。
- 阴历以朔望月（29.53天）为基础，月相可预测，但季节每年漂移约11天。
- 阴阳合历（中国农历）同时追踪太阳和月亮，通过19年7闰的置闰规则维持季节稳定。
- 农历不是阴历，准确应称为阴阳合历；其中的节气部分是纯阳历成分。

## 【英文化改写提示】

- 回归年 → tropical year（approximately 365.2422 days）
- 闰年 → leap year（in lunisolar context: a year with 13 months）
- 闰月 → intercalary month / leap month
- 19年7闰 → the Metonic cycle（19-year cycle with 7 leap months）
- 中气 → principal term（one of the 12 even-numbered solar terms）
- 朔日 → new moon day（the first day of each lunar month）

在英文讲解中注意区分：
- "leap year" in the Gregorian calendar = 366 days (adding Feb 29)
- "leap year" in the Chinese lunisolar calendar = 13 months (adding a leap month)
这两种"闰"完全不同，翻译时需要特别说明。

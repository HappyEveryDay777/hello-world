## 关于本课程

本系列以**中华文化传承者**的视角，系统讲述**中国传统历法**的来龙去脉。

中国历法，是中华文明数千年智慧的结晶。它不只是一套计算日期的工具，更是中国人理解宇宙、感知四季、安排生活、传承文化的根本依据。从天子颁布历法、农人依节气耕种，到普通家庭按历法过节祭祖，历法渗透于中国人生活的每一个角落。

本课程希望做到：用清晰、生动的语言，让每一位读者——无论大人还是孩子——都能真正理解中国历法，而不只是背几个名词。

学完本系列，你将能够：

1. 解释为什么人类需要历法，以及各种历法类型的基本原理
2. 理解中国传统历法（农历）的阴阳合历本质
3. 掌握月份、闰月、年份的安排逻辑
4. 认识二十四节气及其对农业、节庆、生活的意义
5. 入门了解天干地支与生肖文化
6. 了解中国历法从先秦到今天的演变历程
7. 在现代生活中找到传统历法的实际意义

## 关于中国历法的一句话介绍

中国传统历法（通称"农历"或"旧历"），是一套以月亮周期为基础、以太阳周期校正的**阴阳合历**。它既有阴历的特征（以月相计月），又有阳历的特征（用节气锚定太阳位置），配合干支纪年，形成了一套完整、严密、包含丰富文化内涵的时间体系。

这套体系，是中国农耕文明数千年生产实践与天文观测的共同结晶，也是中华民族与天地自然共处方式的活生生记录。

## 关于本课程

颜色，是人类感知世界最直接的方式之一。而中国人对颜色的感知与命名，历经数千年，形成了一套独特而迷人的**传统色彩体系**。

中国传统色，不只是几种颜色，而是一个将**自然、文化、哲学、诗词、器物**融为一体的美学体系。每一个颜色名字，背后都是一个故事，一段历史，一种生活哲学。

比如：
- "月白"——不只是白色，是月光照在白绸上的那种带着蓝意的白
- "天水碧"——不只是绿色，是南唐后主李煜宫中专属染制的颜色，如春水初生
- "胭脂"——不只是红色，是由西域传来的胭脂虫制成的珍贵颜料，曾是贵族专属

本课程系统讲解中国传统色彩的文化内涵、美学价值、历史背景，以及在现代生活中的应用灵感。

## 课程目标

学完本课程后，你将能够：

1. **了解中国传统色彩体系的基本构成**：五色正色、间色、传统色命名逻辑
2. **感受颜色与四季的深层联系**：每个季节的代表色及其文化意涵
3. **欣赏颜色在诗词、绘画、服饰中的运用**：读懂中国艺术中的色彩语言
4. **认识器物中的传统色**：陶瓷、漆器、丝绸中的色彩美学
5. **将传统色彩灵感应用于现代生活**：设计、搭配、创作中的传统色运用

## 关于中国传统色的独特价值

世界各民族都有自己的色彩文化，但中国传统色彩体系的独特之处在于：

**1. 命名的诗意性**
西方颜色名称多来自物理光学（赤橙黄绿蓝靛紫），而中国传统色名多来自自然物象：

- 来自植物：桃花粉、荷叶绿、竹月色
- 来自矿物：石青、朱砂红、铅白
- 来自自然现象：月白、天水碧、霜色
- 来自器物：漆红、釉绿、青花蓝

每一个名字，都是一首微型诗。

**2. 哲学的融入性**
中国传统色彩深受阴阳五行哲学影响：
- 五色（青、赤、黄、白、黑）对应五行（木、火、土、金、水）
- 对应五方（东、南、中、西、北）
- 对应四季和五种情感

颜色在中国文化中，不只是视觉符号，更是宇宙秩序的投影。

**3. 历史的厚重性**
每种传统色的背后，都有历史故事：帝王专属的颜色、宫廷秘制的染方、文人赋予的精神内涵……中国传统色，是一部用颜色写成的历史书。

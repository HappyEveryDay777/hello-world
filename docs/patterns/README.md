## 关于本课程

欢迎来到**中国纹样课**！

你有没有注意到，故宫里那些金碧辉煌的屏风上，盘绕着什么样的龙纹？婶婶出嫁时穿的红嫁衣上，绣着什么样的花？春节时贴在窗户上的剪纸，又藏着哪些吉祥的秘密？

这些美丽的图案，有一个共同的名字——**中国传统纹样**。

纹样，不只是好看的装饰。每一个纹样背后，都藏着一段历史、一个故事、一种中国人对美好生活的深情向往。从新石器时代先民在陶罐上刻下第一条鱼形，到今天的国潮设计师将传统元素融入现代服装，这条美丽的脉络，绵延了将近一万年。

本课程以**中国文化传承人**的视角，带你和孩子一起，踏上一段穿越时空的美丽旅程——从远古到现代，从器物到服饰，从图案到内涵，系统认识中国传统纹样的前世今生。

## 每章的内容结构

为保持课程的整体一致性，每章均包含以下板块：

1. **【开篇故事】** — 用一个生动的场景或故事，带你进入那个时代的纹样世界
2. **【时代背景】** — 简要介绍该历史时期的社会与文化特点
3. **【代表纹样详解】** — 重点介绍2-4种代表性纹样的形态、来源与寓意
4. **【纹样的应用场景】** — 这些纹样出现在哪些器物、服饰、建筑上
5. **【纹样背后的故事】** — 与纹样相关的神话传说、民间故事或历史典故
6. **【审美特点与文化内涵】** — 这一时期纹样的整体风格与文化意义
7. **【课程小结】** — 本章要点总结，便于回顾与记忆
8. **【亲子共读问题】** — 引导大人和孩子一起思考与互动的问题
9. **【视频改编建议】** — 本章内容转化为视频课程的创作思路

## 中国纹样简史速览

在正式开课之前，先来看一张**纹样演变大全图**（文字版），让你对中国纹样的整体面貌有一个宏观印象：

```
新石器时代（约10000-2000 BC）
  └── 陶器上的鱼纹、鸟纹、蛙纹、几何纹
      象征：生殖崇拜、自然崇敬、部落图腾

商代（约1600-1046 BC）
  └── 青铜器上的饕餮纹、云雷纹、夔龙纹
      象征：神权威严、天地沟通、祖先崇拜

西周（约1046-771 BC）
  └── 窃曲纹、重环纹、长尾鸟纹
      象征：礼制秩序、宗法制度

东周（约770-221 BC）
  └── 蟠螭纹、狩猎纹、宴乐纹
      象征：贵族生活、自然崇拜减弱，世俗化加强

汉代（206 BC-220 AD）
  └── 云气纹、四神（青龙白虎朱雀玄武）、菱形纹
      象征：升仙信仰、天地宇宙观、辟邪护佑

魏晋南北朝（220-589）
  └── 忍冬纹（卷草纹）、莲花纹
      象征：佛教传入，天上净土的向往

隋唐（581-907）
  └── 宝相花、联珠纹、卷草纹、团花纹
      象征：国际交流、雍容华贵、自信开放

宋代（960-1279）
  └── 折枝花纹、莲花纹、鱼纹
      象征：文人雅致、自然写实、极简美学

元代（1271-1368）
  └── 缠枝花卉、波斯风格纹样
      象征：多元融合、草原与农耕文明交汇

明代（1368-1644）
  └── 龙凤纹、缠枝莲、暗八仙、补子纹
      象征：皇权鼎盛、吉祥文化系统化

清代（1644-1912）
  └── 百子图、八宝纹、十二章纹、岁寒三友
      象征：吉祥祈福登峰造极、精工细作

近现代（1912至今）
  └── 传承与创新：国潮设计、非遗复兴
      象征：文化自信、东方美学的当代表达
```

## 关于纹样的三个问题（先想一想）

在开始学习之前，不妨先问问自己，或者问问你身边的孩子：

1. 你见过哪些让你觉得"很中国"的图案？
2. 你知道故宫的墙上、皇帝的衣服上，为什么要画龙吗？
3. 中国人为什么这么喜欢在东西上刻花、绣花、画花？

带着这三个问题，开始我们的纹样之旅吧！

## 中国纹样核心词汇对照表

| 中文 | 拼音 | 英文 | 简要说明 |
|------|------|------|---------|
| 纹样 | wén yàng | pattern / motif | 装饰性图案的总称 |
| 饕餮纹 | tāo tiè wén | taotie pattern | 商代青铜器上的兽面纹样 |
| 云雷纹 | yún léi wén | cloud-thunder pattern | 几何形漩涡纹，是中国最古老的纹样之一 |
| 龙纹 | lóng wén | dragon pattern | 中国文化最重要的神圣纹样 |
| 凤纹 | fèng wén | phoenix pattern | 代表高贵与吉祥，常与龙配对 |
| 宝相花 | bǎo xiàng huā | treasure flower | 唐代将莲花、牡丹等融合的理想化花卉纹样 |
| 卷草纹 | juǎn cǎo wén | scrolling vine pattern | 以植物藤蔓为基础的连续纹样 |
| 吉祥纹样 | jí xiáng wén yàng | auspicious patterns | 含有美好寓意的纹样体系 |

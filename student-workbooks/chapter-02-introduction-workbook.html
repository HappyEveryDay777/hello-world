Student Workbook

第 2 章：你好，我是谁——基础交际入门 · 学生作业册版

面向学生的作业与学习记录页，帮助完成预习、课堂任务、课后作业、自评与提交。

第一阶段：入门基础 第3—4周 使用方式：预习 + 课中 + 课后

一、学生使用说明

请先阅读本章目标，再按“预习—课堂—课后—提交”的顺序使用本作业册。遇到不明白的地方，可回到章节总页或练习题版。

二、学习流程清单

  • 先阅读本章目标，确认本次学习要完成什么。
  • 预习核心词汇：我、你、他、她、我们、你们、名字、叫
  • 课堂上完成口语练习：两人互问姓名、国籍和语言；替换练习：我 / 你 / 他 + 是……。
  • 课后完成写作任务：抄写人称代词并造句；写 5 句“我是……”句子。
  • 最终提交：完成一段 1 分钟自我介绍，内容包括姓名、国籍、职业 / 身份和语言情况。

三、课前预习

本章目标

  • 学会使用人称代词介绍自己和他人。
  • 掌握“是”字句和基础特殊疑问句。
  • 能在课堂或初次见面场景中完成简单自我介绍。

预习词汇

  • 我
  • 你
  • 他
  • 她
  • 我们
  • 你们
  • 名字
  • 叫
  • 是
  • 学生

四、课堂任务记录

口语任务

  • 两人互问姓名、国籍和语言。
  • 替换练习：我 / 你 / 他 + 是……
  • 课堂表达操练：我明白了、我不知道、请问。
  • 完成“认识新同学”角色扮演。

阅读任务

大家好！我叫李安。我是学生，我是英国人。我会说英语，也学习汉语。她叫玛丽，她是我的朋友。

  • 李安是什么身份？
  • 他会说什么语言？
  • 文中“她”指谁？

五、课后作业

书写与作业

  • 抄写人称代词并造句。
  • 写 5 句“我是……”句子。
  • 填写一张中文个人信息卡。
  • 用 6—8 句写一段自我介绍。

复习检测

  • 词汇测验：代词、职业、国籍。
  • 口头问答：姓名、职业、语言。
  • 完成一段同学互相介绍。
  • 修改自我介绍中的“是”字句错误。
  • 录制 1 分钟自我介绍。

六、自我检查与提交

自我检查

  • 我是否能做到：能说出“我叫……”“我是……”。
  • 我是否能做到：能问“你叫什么名字？”“你是哪国人？”
  • 我是否能做到：能完成约 1 分钟的个人介绍。

提交说明

  • 完成本章作业册空白内容并整理成可提交版本。
  • 附上课堂练习记录、录音或书写照片（如适用）。
  • 提交前根据自我检查项勾选完成情况。
  • 完成一段 1 分钟自我介绍，内容包括姓名、国籍、职业 / 身份和语言情况。

Bilingual Teaching Script

第 6 章：汉字入门与书写规则 · 中英双语教学稿版

适合教师边上课边使用的课堂话语稿，按环节提供中文教学话语与英文支持说明。

第二阶段：生活交流 第11—12周 课堂语言：中文主导 + 英文辅助

课堂使用原则

  • 先用中文发出课堂指令，再用英文补充活动要求。
  • 英文说明重点服务于难点澄清，不替代中文操练。
  • 每个环节都先示范，再让学生模仿与扩展。
  • 课堂结尾优先让学生用中文总结，再由教师用英文收束。

双语教学流程稿

开场导入

中文教学稿

同学们好，今天我们学习“汉字入门与书写规则”。先一起复习上节课的内容。

English support

Good morning everyone. Today we will study this chapter topic. Start with a short review before new input.

学生预期输出

学生能回应问候，并说出 2—3 个已学相关词汇：一、二、三。

词汇呈现

中文教学稿

请看这些词：一、二、三、十、口、日。跟我读一遍。

English support

Model each word first, then check meaning with quick examples instead of long translation.

学生预期输出

学生跟读并能识别重点词汇，尝试用其中 2 个词造短句。

句型讲解

中文教学稿

今天的重点句型是：这个字怎么写？；请写一写这个字。；这个字是什么意思？。

English support

Use English briefly to explain function, word order, and when to use each pattern.

学生预期输出

学生能模仿句型并完成替换操练。

引导操练

中文教学稿

看字读音训练。

English support

Give one full model in Chinese, then use English only if learners cannot follow the task instructions.

学生预期输出

学生在两人或小组内完成口头练习，并尝试自然回应。

阅读指导

中文教学稿

现在请读这段短文：我学习汉字。老师先教我们写“人”“口”“山”“水”。这些字不长，可是要注意笔顺。写得慢一点，字会更清楚。

English support

Pre-teach only the difficult words. Ask comprehension questions in simple Chinese first, then support in English if needed.

学生预期输出

学生能回答阅读问题：老师先教了哪些字？；写汉字时要注意什么？；为什么要写得慢一点？。

输出与收束

中文教学稿

最后请完成今天的输出任务：书写一组基础汉字，并完成一次拼音到汉字的简单认读与书写练习。

English support

Summarize the lesson in English only after students first try to reflect in Chinese.

学生预期输出

学生完成个人输出、同伴展示或课堂汇报。

重点提醒

重点词汇

一、二、三、十、口、日、月、人

重点句型

这个字怎么写？；请写一写这个字。；这个字是什么意思？

英语讲解重点

  • Explain stroke order as a memory system, not an arbitrary rule list.
  • Use English to compare “character” and “word” because learners often confuse them.
  • Clarify radicals as meaning clues and components as structural parts.
  • Provide feedback on spacing, balance, and stroke direction.

Bilingual Teaching Script

第 3 章：数字、时间与日期 · 中英双语教学稿版

适合教师边上课边使用的课堂话语稿，按环节提供中文教学话语与英文支持说明。

第一阶段：入门基础 第5—6周 课堂语言：中文主导 + 英文辅助

课堂使用原则

  • 先用中文发出课堂指令，再用英文补充活动要求。
  • 英文说明重点服务于难点澄清，不替代中文操练。
  • 每个环节都先示范，再让学生模仿与扩展。
  • 课堂结尾优先让学生用中文总结，再由教师用英文收束。

双语教学流程稿

开场导入

中文教学稿

同学们好，今天我们学习“数字、时间与日期”。先一起复习上节课的内容。

English support

Good morning everyone. Today we will study this chapter topic. Start with a short review before new input.

学生预期输出

学生能回应问候，并说出 2—3 个已学相关词汇：零、一、二。

词汇呈现

中文教学稿

请看这些词：零、一、二、两、三、四。跟我读一遍。

English support

Model each word first, then check meaning with quick examples instead of long translation.

学生预期输出

学生跟读并能识别重点词汇，尝试用其中 2 个词造短句。

句型讲解

中文教学稿

今天的重点句型是：今天几月几号；现在几点；我……岁。

English support

Use English briefly to explain function, word order, and when to use each pattern.

学生预期输出

学生能模仿句型并完成替换操练。

引导操练

中文教学稿

听写电话号码并复述。

English support

Give one full model in Chinese, then use English only if learners cannot follow the task instructions.

学生预期输出

学生在两人或小组内完成口头练习，并尝试自然回应。

阅读指导

中文教学稿

现在请读这段短文：今天是五月十二号，星期一。我早上七点起床，八点上课。下午三点半我去图书馆，晚上十点睡觉。

English support

Pre-teach only the difficult words. Ask comprehension questions in simple Chinese first, then support in English if needed.

学生预期输出

学生能回答阅读问题：今天是星期几；他几点上课；下午三点半以后他去哪里。

输出与收束

中文教学稿

最后请完成今天的输出任务：介绍自己的生日、年龄和每天的基本作息时间。

English support

Summarize the lesson in English only after students first try to reflect in Chinese.

学生预期输出

学生完成个人输出、同伴展示或课堂汇报。

重点提醒

重点词汇

零、一、二、两、三、四、五、六

重点句型

今天几月几号；现在几点；我……岁。

英语讲解重点

  • Explain the Chinese big-to-small order for dates and clock expressions.
  • Highlight the usage difference between 二 and 两.
  • Use English to explain that Chinese does not require prepositions like “at” before every clock time.
  • Correct number grouping when learners read long phone numbers.

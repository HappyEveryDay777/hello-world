Unified Print Template

可直接打印的统一封面与页眉页脚版

提供整套资料统一使用的打印封面、页眉、页脚和内页样式示意，可直接打开浏览器打印，也可作为导出 PDF 的统一模板。

适用：教师讲义 / 学生作业册 输出：打印 / PDF 样式：统一封面 / 页眉 / 页脚 / 内页

一、模板使用说明

封面用途

作为整套教师讲义或整套学生作业册的统一封面，保证资料视觉一致。

页眉用途

标识课程名称、章节名称和资源类型，方便打印后快速翻阅。

页脚用途

统一标识阶段、页码位置、使用说明和归档信息。

二、打印检查清单

  • 打印前确认浏览器缩放为 100%，并启用背景图形。
  • 按需要替换封面中的资料名称、章节范围和日期信息。
  • 导出 PDF 前确认页眉、页脚、页码和边距位置一致。
  • 教师讲义与学生作业册可共用此模板，只需替换标题和副标题。

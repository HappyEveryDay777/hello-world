# 二十四节气 · 中华时令文化精粹

> 春雨惊春清谷天，夏满芒夏暑相连，秋处露秋寒霜降，冬雪雪冬小大寒。
> ——《二十四节气歌》

---

## 什么是二十四节气？

二十四节气，是中国古代先民在长期农耕生活中，通过观察太阳在黄道上的位置变化，总结出的一套时令体系。它将一年365天细分为二十四个节点，每个节气约相隔15天，完整记录了一年之中天地自然的运行规律。

这套体系诞生于黄河流域，距今已有两千余年的历史。公元前104年，《太初历》正式将二十四节气纳入历法，从此成为中国农业社会的时间坐标。2016年，联合国教科文组织将"二十四节气——中国人通过观察太阳周年运动而形成的时间知识体系及其实践"列入人类非物质文化遗产代表作名录，向世界宣告了这一智慧结晶的普世价值。

二十四节气不仅是农耕的指南针，更是中国人感知自然、顺应天地、调养身心的生活哲学。它蕴含着中华文明"天人合一"的核心理念——人是自然的一部分，应当随四时流转而调整自己的生活节律。

---

## 为什么要了解二十四节气？

在快节奏的现代生活中，我们越来越习惯于恒温空调、全年供应的蔬果和24小时亮灯的城市。我们正在慢慢忘记：窗外的世界，正在随着时令悄悄改变。

重新认识二十四节气，是重新与自然建立联结的方式。它提醒我们：

- **春天**，要顺应生发之气，舒展身心，播种希望；
- **夏天**，要拥抱阳气鼎盛，尽情生长，释放活力；
- **秋天**，要顺应收敛之道，滋养内敛，积蓄能量；
- **冬天**，要遵循潜藏之理，保暖藏精，静待复苏。

这套古老的节律，在今天依然有深刻的现实意义。

## 四季节气总览

### 🌱 春季六节气 — 生发、复苏、蓬勃

春天是一年之始，是大地从沉睡中苏醒的季节。六个春季节气像六个乐章，从第一声春雷，到最后一场春雨，奏响了生命复苏的交响曲。

| 节气 | 含义 | 大约时间 |
|------|------|----------|
| 立春 | 春天开始，万象更新 | 2月3-5日 |
| 雨水 | 降水增多，冰雪消融 | 2月18-20日 |
| 惊蛰 | 春雷惊醒冬眠生灵 | 3月5-7日 |
| 春分 | 昼夜平分，春色正中 | 3月20-22日 |
| 清明 | 天清气明，慎终追远 | 4月4-6日 |
| 谷雨 | 雨生百谷，播种时节 | 4月19-21日 |

---

### ☀️ 夏季六节气 — 繁茂、热烈、充盈

夏天是生命最饱满的季节，阳气升至顶点，万物蓬勃生长。六个夏季节气记录了从初夏的清新，到盛夏的炽热，再到暑末的转折。

| 节气 | 含义 | 大约时间 |
|------|------|----------|
| 立夏 | 夏天开始，万物茁壮 | 5月5-7日 |
| 小满 | 麦粒初熟，物候小满 | 5月20-22日 |
| 芒种 | 有芒之谷可种，忙碌农时 | 6月5-7日 |
| 夏至 | 白昼最长，阳气极盛 | 6月21-22日 |
| 小暑 | 暑热渐至，未至极盛 | 7月6-8日 |
| 大暑 | 一年最热，暑气鼎盛 | 7月22-24日 |

---

### 🍂 秋季六节气 — 收获、转折、沉淀

秋天是大地的收获季，也是自然由盛转衰、由外向内的转折点。六个秋季节气从初秋的余热，经历白露的清凉，走向霜降的肃杀。

| 节气 | 含义 | 大约时间 |
|------|------|----------|
| 立秋 | 秋天开始，暑气渐退 | 8月7-9日 |
| 处暑 | 暑热止息，天气转凉 | 8月22-24日 |
| 白露 | 露珠凝结，秋意渐浓 | 9月7-9日 |
| 秋分 | 昼夜平分，秋色正中 | 9月22-24日 |
| 寒露 | 露水更凉，秋深叶黄 | 10月7-9日 |
| 霜降 | 初霜降临，草木枯黄 | 10月23-24日 |

---

### ❄️ 冬季六节气 — 收藏、静养、积蓄

冬天是自然归于静默的季节，万物潜藏，等待来年春天的重生。六个冬季节气从初冬的寂寥，到大雪的银装素裹，再到冬至的阴极阳生，构成了生命轮回最深邃的乐章。

| 节气 | 含义 | 大约时间 |
|------|------|----------|
| 立冬 | 冬天开始，万物收藏 | 11月7-8日 |
| 小雪 | 初雪飘落，寒意渐生 | 11月22-23日 |
| 大雪 | 大雪纷飞，封冻大地 | 12月6-8日 |
| 冬至 | 白昼最短，阴极阳生 | 12月21-23日 |
| 小寒 | 寒气渐重，尚未至极 | 1月5-7日 |
| 大寒 | 一年最冷，寒气极盛 | 1月19-21日 |
